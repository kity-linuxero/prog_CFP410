Repaso de la clase anterior

Repaso: Partes de una computadora

  • CPU:
    • Procesa la información
  • Memoria (RAM):
    • Los programas que están corriendo en un momento determinado. Variables, objetos, etc.
  • Almacenamiento:
    • Donde se guardan los archivos. Los programas también, pero no es lo que está en ejecución.
  • Periféricos:
    • Como interactuamos con la computadora. Teclado, monitor... etc.

Repaso: ¿Qué es programar?

  • Programa:
    • Un programa es un conjunto de instrucciones, ejecutables sobre una computadora, que permite cumplir una function específica.
  • Pseudocódigo:
    • Una forma "humana" de describir una secuencia de pasos que luego se traducirá a un programa.
  • Lenguajes de programación:
    • Una "forma" de escribir nuestros programas. Hay distintos tipos y para distintos propósitos.
  • Librerías:
    • Es código escrito por terceros que extiende las funcionalidades de las librerías estándar del lenguaje.

Repaso: Algoritmos

  • Algoritmo:
    • Es la especificación de la secuencia de instrucciones a realizar sobre una computadora para alcanzar un resultado deseado.

Repaso: Estructuras de control

  • Secuencia:
    • Las instrucciones se ejecuta una tras otra en orden de arriba hacia abajo. Ref
  • Decisión (if):
    • Altera el flujo secuencial en base a una condición. Ref
  • Decisión múltiple (case/switch):
    • Útil cuando hay muchas condiciones. Ref
  • Iteración (while/for):
    • Se usa para ejecutar un bloque mas de una vez. Ref

Repaso: Estructuras y datos

  • Instrucciones:
    • Las instrucciones se ejecuta una tras otra en orden de arriba hacia abajo. Ref
    • Asignación
    • Decisión
    • ...
  • Datos:
    • Son valores de información que disponemos y podemos modificar. Ref

Tipos de datos

Tipos de datos

En los programas es necesario representar información del mundo real como números, letras, listas de nombres, etc.

Para poder representarlos aparecen los datos y los tipos de datos.

Tipos de datos

Los tipos de datos se caracterizan por:

  • Un rango de valores posibles
  • Un conjunto de operaciones realizables sobre ese tipo
  • Su representación interna

Tipos de datos simples

Definiremos como tipos de datos simples a los:

  • Numéricos:
    • Enteros (-n,...-1,0,1....n)
    • Reales (-n,...-0.000001...,0.0,...,0.1,...n)
  • Lógico:
    • Verdadero
    • Falso
  • Caracter o lista de caracteres
    • Char: un solo caracter: ('a','b','c'...'z'...'&'...)
    • String: cadena de caracteres: ("aaa","aab","Hola"...)

Operaciones sobre tipos

  • Es posible aplicar operaciones para los distintos tipos de datos.
  • No en todos los casos será igual y el resultado puede variar dependiendo del tipo de datos usado.

Operaciones sobre tipos

Sumar dos enteros:


							1+1

El resultado será:


								2

Sumar dos reales:


							1.23+1.9

El resultado será:


							3.13

Operaciones sobre tipos numéricos

Hay muchísimas operaciones posibles. Vamos a mencionar las básicas:

  • Suma (+)
  • Resta (-)
  • Multiplicación (*)
  • División (/)
  • División entera (//) - Sin decimales
  • Módulo (%) - El resto de la división

Tipo de dato lógico

También llamado booleano o bool. Es un dato que puede tomar un valor entre un conjunto formado por dos posibles:

  • Verdadero (true)
  • Falso (false)

Operaciones sobre booleanos

Los operadores booleanos básicos son:

  • Negación (not)
  • Suma o disyunción (or)
  • Producto o conjunción (and)

A continuación veremos ejemplos de lógica.

Operaciones sobre booleanos

NOT

p not p
Verdadero (True) Falso (False)
Falso (False) Verdadero (True)

Básicamente invierte el valor.

Operaciones sobre booleanos

AND

p q p and q
True True True
True False False
False True False
False False False

Si hay un valor en false, el resultado será siempre false.

Operaciones sobre booleanos

OR

p q p or q
True True True
True False True
False True True
False False False

Si hay valor en true, el resultado será siempre true.

Comparación entre tipos

Es posible hacer operaciones de comparación sobre los datos. La comparación nos dará como resultado un valor booleano (verdadero o falso).

  • Comparación (==)
    • Compara si dos datos del mismo tipo tienen el mismo valor.
  • Mayor que (>)
    • Verifica si un valor es mayor a otro (se lee desde izquierda a derecha)
  • Menor que (<)
    • Verifica si un valor es menor a otro (se lee desde izquierda a derecha)
  • Distinto que (!=)
    • Verifica si los valores a comparar son distintos

Para poder realizar las comparaciones los datos tienen que ser del mismo tipo o compatibles con la operación.

Ejemplos de comparación

Comparar dos números:


								1==2	# False
								1==1 	# True
								233<12	# False
								210>1	# True
								210!=210 # False
								2.1<2	# False. Compatibilidad entre tipos entero y real

Comparaciones en otro tipo de datos:


								"Juan"=="Juan" # True
								"A"<"B" # True
								"A"<"a" # True
								"Cristian" != "Beto" # True

Operaciones sobre string

Sumar dos caracteres:


							'1'+'1'

El resultado será:


							'11'

¿Por qué '1'+'1' = '11'?. Porque '1' NO es un número, sino es un caracter (ver las comillas). Si sumamos dos caracteres o cadena de caracteres en realidad los estamos "concatenando".

Operaciones sobre string

"Sumar" o concatenar cadenas de caracteres puede ser muy util.

Ejemplo:


								nombre= "Juan"
								apellido= "Perez"
								print(nombre+apellido)

El resultado será:


								JuanPerez

De ahora en mas llamaremos a las cadenas de caracteres como String.

Operaciones sobre String

Podemos concatenar muchos strings

Ejemplo:


								nombre= "Juan"
								apellido= "Perez"
								print("¡Hola " + nombre + " " + apellido + "!")

El resultado será:


								¡Hola Juan Perez!

Recomendación: Prueben copiar y pegar el código de mas arriba en el intérprete Python Online y jueguen cambiando el orden, agregando mas strings, palabras... lo que se les ocurra para entender como se comporta el concatenado de strings.

Operaciones sobre String

Podemos imprimir string de diferentes maneras:

Para no agregar un espacio podemos usar ,


								nombre= "Juan"
								apellido= "Perez"
								print("¡Hola",nombre,apellido + "!")

El resultado será el mismo:


								¡Hola Juan Perez!

Operaciones sobre String

Para no concatenar por fuera del string:


								nombre= "Juan"
								apellido= "Perez"
								print(f"¡Hola {nombre} {apellido}!")

El resultado será:


								¡Hola Juan Perez!

Operaciones sobre String

print(f") puede ser muy útil:


								a= 23
								b= 34
								print(f'La suma de {a} + {b} = {(a+b)}')

El resultado será el mismo:


								La suma de 23 + 34 = 57
con print(f'') NO es necesario convertir a string con "str" la suma de los números.

Conversión de tipos

  • Puede presentarse la necesidad de convertir tipos. A veces esto es posible pero otras veces no.
  • Una de las conversiones mas comunes es convertir números a string y viceversa.
  • Existen instrucciones especiales para convertir tipos

Conversión de tipos

Conversión de número a string

Se usa la instrucción str


							>>> str(2)
							'2' # Resultado

Conversión de string a número entero

Se usa la instrucción int


							>>> int('2')
							2 # Resultado

Conversión de tipos

Conversión de número a string

Se usa la instrucción str


							var1= 233 # Se guarda en var1 un número entero
							var2= str(var1) # Se convierte a string y se guarda en var2

Conversión de string a número entero

Se usa la instrucción int


							var3= '512' # Se guarda en var3 un string.
							var4= int(var3) # Se convierte a entero y se guarda en var4
